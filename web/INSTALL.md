# Anthuan.Env Web App

A modern, interactive guide for your macOS web dev setup.

## Quick Start

1. **Go to the web project folder:**
   ```bash
   cd ~/Sites/Anthuan.Env/web
   ```

2. **Install dependencies:**
   *(We recommend using `pnpm`, but `npm` works too)*
   ```bash
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   pnpm run dev
   ```

4. **Open in your browser:**
   Navigate to [http://localhost:5173](http://localhost:5173)
